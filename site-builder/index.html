SPEC v2.3 + Google 公式基準 + Cornell GEO 論文の 3法規を system prompt 全文埋込で常時参照。長文コンテキスト処理で最高性能。

#3

プロンプトキャッシュ対応

Anthropic 公式の ephemeral cache で 3法規部分をキャッシュ化。2回目以降は最大 85% トークン費用削減。

設計思想: 本アプリは設計当初から Claude Opus 4.6 専用として構築されています。S-RANK 品質保証を数値的に成立させる3条件（日本語E-E-A-T実測値 / ephemeral cache 90%割引 / 3法規40KB追従）が Opus 4.6 に最適化されているためです。Gemini / GPT は強力な汎用AIですが、本アプリの設計前提には未組込です。Claude Sonnet 4.5 は routingMode=balanced 時のサブモデルとして利用します。
