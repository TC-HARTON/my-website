HARTON 代表

静岡県沼津市を拠点に、WEB制作・業務自動化・AI売上予測の3分野で中小企業のDX推進を支援。「モヤっとを可視化し、淀みない経営へ」をビジョンに、テクノロジーの力で地域企業の課題解決に取り組んでいます。

専門分野・技術スキル

WEB制作

HTML/CSS/JavaScript、Tailwind CSS、SEO最適化、LLMO対応、Core Web Vitals最適化。レスポンシブ対応の高速サイト構築。

業務自動化

Python、Google Apps Script、Excel VBA/マクロ。請求書自動生成、データ集計、定期レポートメール送信等の自動化。

AI売上予測

Python、scikit-learn、TensorFlow。過去データからの売上予測・需要予測モデル構築。ダッシュボードでの可視化。

実績・一次情報

MQL4/MQL5による自動売買システム開発

MetaTrader向け自動売買エキスパートアドバイザー（EA）を複数開発。市場データの統計分析から売買ロジックの設計、バックテスト検証、本番稼働まで一貫して実施。リアルタイムデータ処理とリスク管理の知見を蓄積。

中小企業向けDX支援の実践知見

沼津市を中心とした静岡県東部の中小企業に対し、業務フローのヒアリングから課題特定、自動化ソリューションの提案・実装までをワンストップで提供。IT専門用語を使わず、現場目線のコミュニケーションで導入障壁を低減。

SEO+LLMO完全最適化サイト構築

Google検索セントラルガイドライン、E-E-A-T基準、Core Web Vitals最新仕様に完全準拠したサイト構築メソドロジーを独自に確立。HARTON自社サイトにてSクラス判定を取得し、そのノウハウをクライアントサイト構築に展開。
