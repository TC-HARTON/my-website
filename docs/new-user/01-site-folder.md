# サイトフォルダの選び方

## このステップで行うこと
WEBサイトのファイル（HTML・CSS・画像など）を保存する場所を決めます。

## サイトフォルダとは？
パソコン上の「フォルダ」のことです。WEBサイトは複数のファイルで構成されており、それらをまとめて管理する場所が必要です。

たとえば：
- `デスクトップ > my-site` というフォルダを作り、そこにサイトのファイルを置く
- すでにサイトのファイルがある場合は、そのフォルダを指定する

## 操作手順

### 新しくサイトを作る場合
1. 「新しくサイトを作る」を選択
2. 「📁 参照」ボタンをクリック
3. フォルダ選択画面が開く（デスクトップが表示されます）
4. 新しいフォルダを作るか、空のフォルダを選択
5. 「次へ →」をクリック

### 既存サイトを取り込む場合
1. 「既存サイトを取り込む」を選択
2. 「📁 参照」ボタンをクリック
3. サイトのHTMLファイルがあるフォルダを選択
4. 「次へ →」をクリック → サイト情報が自動的に読み取られます

## よくある質問

### Q. サーバー上にあるサイトはどうすればいい？
レンタルサーバーやVPSにあるサイトは、まずFTPソフト等でパソコンにダウンロードしてください。ダウンロードしたフォルダを指定すれば、サイト情報を読み取れます。

### Q. どこにフォルダを作ればいい？
デスクトップや「ドキュメント」フォルダが分かりやすくおすすめです。フォルダ名は半角英数字（例: `my-site`、`company-website`）にすると安全です。

### Q. フォルダ名に日本語を使っても大丈夫？
動作はしますが、サーバーへのアップロード時に問題が出る場合があります。半角英数字のフォルダ名を推奨します。

### Q. GitHubで管理しているサイトは？
GitHubからクローンしたフォルダをそのまま指定できます。Site Builderで編集後、Git・デプロイ機能でそのままプッシュできます。
