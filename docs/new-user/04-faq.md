# FAQ登録ガイド

## このステップで行うこと
よくある質問と回答を登録します。最低5問以上が必要です（SPEC v2.3要件、GEO観点では8〜10問推奨）。

## なぜFAQが重要？
- **Google検索のリッチリザルト**: FAQ構造化データにより、検索結果にQ&Aが直接表示されます
- **ユーザーの疑問を先回り**: 問い合わせ前に回答が得られ、コンバージョン率が向上
- **SEOコンテンツ量**: 質の高いFAQはページのコンテンツ評価を高めます

## SPEC要件
- **最低5問以上**（5問未満ではSPECチェックでFAILになります）
- 質問・回答ともに具体的な内容であること

## 効果的なFAQの書き方

### 質問のコツ
- ユーザーが実際に検索しそうなフレーズで書く
- 「〜とは？」「〜の費用は？」「〜にかかる期間は？」が定番
- 事業固有の具体的な質問を含める

### 回答のコツ
- 最初の1文で端的に回答し、補足説明を続ける
- 100〜200文字程度が適切
- 数字や具体例を含めると信頼性が上がる

## よくある質問パターン（業種共通）
1. サービスの料金・価格帯
2. 対応エリア・地域
3. 納期・所要時間
4. 相談・見積もりの方法
5. 他社との違い・強み
