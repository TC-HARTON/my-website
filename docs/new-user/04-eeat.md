# E-E-A-T著者情報の設定

## このステップで行うこと
サイトの著者（あなた）の専門性・経験・権威性・信頼性を示す情報を設定します。

## E-E-A-Tとは？
Googleがサイトの品質を評価する基準です：
- **Experience（経験）** — 実体験に基づくコンテンツか
- **Expertise（専門性）** — 専門知識があるか
- **Authoritativeness（権威性）** — その分野で認められているか
- **Trustworthiness（信頼性）** — 信頼できる情報源か

## 入力項目

### 著者基本情報
- **著者名**: サイトに表示される名前
- **資格・肩書き**: 専門性を示す情報（例：中小企業診断士、IT導入補助事業者）
- **プロフィールページ**: 通常は /profile/（自動設定済み）

### ソーシャルリンク（sameAs）
外部プロフィールのURL。Googleが著者の実在性を確認する材料になります。
- LinkedIn、Twitter/X、GitHub、note、Wantedly など

### 専門分野（knowsAbout）
カンマ区切りで専門分野を入力（例：WEB制作, SEO対策, 業務自動化）

### プロフィール文
- **自己紹介**: 経歴や実績を含む紹介文
- **理念**: 事業に対する考え方や価値観

## 効果
- プロフィールページが自動生成され、全ページからリンクされます
- 構造化データ（Person型JSON-LD）に反映
- ブログ記事の著者情報として表示
