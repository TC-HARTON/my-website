# ドメインとホスティングの設定

> サイトの「住所」にあたるURLを決めましょう

## はじめに

ドメインとは、サイトのURL（例: `harton.co.jp`）のことです。
お店の住所のように、お客様があなたのサイトにたどり着くための大切な情報です。

ドメインがなくてもサイトは作れます。まずは無料で始めて、後から独自ドメインに切り替えることもできます。

## 選択肢

### A) 無料のサブドメインで始める（おすすめ）

費用ゼロですぐに公開できます。後から独自ドメインへの切り替えも可能です。

| サービス | URL形式 | 特徴 |
|---------|---------|------|
| [Cloudflare Pages](https://pages.cloudflare.com/) | yourname.pages.dev | 高速・無料SSL・推奨 |
| [GitHub Pages](https://pages.github.com/) | yourname.github.io | シンプル・GitHub連携 |
| [Netlify](https://www.netlify.com/) | yourname.netlify.app | ドラッグ&ドロップ対応 |
| [Vercel](https://vercel.com/) | yourname.vercel.app | 高機能・無料枠あり |

### B) 独自ドメインを取得する

ビジネス用には独自ドメインがおすすめです。年間1,000〜3,000円程度で取得できます。

| サービス | 特徴 |
|---------|------|
| [お名前.com](https://www.onamae.com/) | 国内最大手・日本語サポート |
| [ムームードメイン](https://muumuu-domain.com/) | 低価格・初心者向け |
| [Cloudflare Registrar](https://www.cloudflare.com/products/registrar/) | 原価提供・最安級 |
| [Squarespace Domains](https://domains.squarespace.com/) | 海外大手・シンプル管理 |

## 設定手順

1. 上記のサービスでドメインを取得（またはサブドメインを決定）
2. Site Builderの「初期設定」→「サイト・ドメイン」にURLを入力
3. 「保存」をクリック

## よくある質問

**Q: 無料と有料、どちらがいいですか？**
A: まずは無料で始めて、サイトが完成してから独自ドメインに切り替えるのがおすすめです。

**Q: 独自ドメインに切り替えるとURLが変わりますか？**
A: はい。ただしリダイレクト設定で旧URLからの転送が可能です。

**Q: .comと.jpの違いは？**
A: .jpは日本の企業・個人向けで信頼感があります。.comは世界共通で汎用的です。ビジネスの対象に合わせて選びましょう。

**Q: 後から設定を変更できますか？**
A: はい。設定画面からいつでも変更できます。
