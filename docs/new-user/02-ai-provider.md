# AIプロバイダーの設定方法

## このステップで行うこと
サイトのコンテンツ（文章・SEOタグ等）をAIで自動生成するために、AIサービスのAPIキーを設定します。

## APIキーとは？
AIサービスを利用するための「鍵」のようなものです。各AIサービスのサイトで無料アカウントを作成すると取得できます。

**3つのプロバイダーから選べます（1つでOK）：**
- **Claude（Anthropic）** — 高品質な日本語生成に強い
- **Gemini（Google）** — 無料枠が大きく始めやすい
- **ChatGPT（OpenAI）** — 幅広い用途に対応

## 操作手順

### 1. APIキーを取得する
各プロバイダー名の隣にある「キー取得」リンクから公式サイトへ移動し、アカウント作成・APIキー発行を行います。

- **Claude**: console.anthropic.com でアカウント作成 → Settings → API Keys → Create Key
- **Gemini**: aistudio.google.com でGoogleアカウントログイン → Get API Key
- **ChatGPT**: platform.openai.com でアカウント作成 → API Keys → Create new secret key

### 2. APIキーを入力して「設定」をクリック
1. 取得したキーをAPI Key欄に貼り付け
2. 「設定」ボタンをクリック → キーが保存されます
3. 自動的に利用可能なモデル一覧が取得されます

### 3. モデルを選択（任意）
デフォルトのモデルが自動選択されますが、好みに応じて変更できます。「更新」ボタンで最新のモデル一覧を再取得できます。

## よくある質問

### Q. どのプロバイダーがおすすめ？
初めての方には**Gemini**が無料枠が大きくおすすめです。品質重視なら**Claude**が日本語生成に優れています。

### Q. 複数のプロバイダーを設定できる？
はい。複数設定して、メインで使うプロバイダーを「AIプロバイダー」ドロップダウンで選択できます。

### Q. APIキーに料金はかかる？
各サービスによります。多くのサービスに無料枠があります。Site Builderでの利用量は通常少量です。

### Q. APIキーは安全？
APIキーはこのPC内のconfig.jsonにのみ保存され、外部に送信されることはありません。AI APIへの通信時のみ使用されます。

### Q. モデル一覧が取得できない
APIキーが正しいか確認してください。「接続テスト実行」ボタンで接続状態を確認できます。
