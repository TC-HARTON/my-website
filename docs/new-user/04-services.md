# サービス登録のコツ

## このステップで行うこと
あなたの事業が提供するサービスを登録します。登録した内容はサイトのサービスセクション・構造化データ・個別ページに反映されます。

## なぜサービス登録が必要？
- **サイトの主要コンテンツ**: トップページのサービスセクションに表示されます
- **SEO効果**: 構造化データ（JSON-LD）にサービス情報が含まれ、検索結果のリッチ表示に貢献
- **個別ページ生成**: 各サービスの専用ページを自動生成できます

## 入力項目

### サービス名
サービスの正式名称（例：WEBサイト制作、業務自動化コンサルティング）

### 説明
サービスの概要（50〜150文字推奨）。検索結果にも表示される可能性があります。

### パス
サービス個別ページのURL（例：/services/web/）。空欄の場合は自動生成されます。

## コツ
- 3〜5つのサービスが理想的（多すぎると焦点がぼやける）
- 各サービスの説明は具体的に — 「何を」「誰に」「どんな価値を」
- パスは短く分かりやすい英語で（例：/services/consulting/）
