# AIチャットボットの設定

> サイト訪問者の質問にAIが24時間自動で回答します

## はじめに

AIチャットボットを設置すると、サイトを訪れたお客様の質問に24時間自動で回答できます。
営業時間外でも、よくある質問への回答やサービスの案内を自動で行います。

## 何ができるか

チャットボットは、あなたが登録した以下の情報をもとに回答します:
- **事業情報**: 会社名、住所、営業時間、連絡先
- **サービス内容**: 提供しているサービスの名前と説明
- **FAQ**: よくある質問と回答

お客様が「営業時間は？」と聞けば営業時間を、「どんなサービスがありますか？」と聞けばサービス一覧を回答します。

## 設定項目

### ON / OFF
チャットボットを表示するかどうかを切り替えます。後からいつでも変更できます。

### 挨拶文
サイト訪問者に最初に表示されるメッセージです。

例:
- 「こんにちは！○○のAIアシスタントです。サービスや料金について何でもお気軽にお聞きください。」
- 「○○へようこそ！ご質問がありましたらお気軽にどうぞ。」

### プロキシURL（上級者向け）
独自のAPIプロキシを使用する場合に設定します。通常は空欄のままで問題ありません。

## 設定手順

1. Site Builderの「初期設定」→「訪問者チャットボット」
2. 「チャットボット有効」をONに設定
3. 挨拶文を入力（AIヒアリングで自動提案もされます）
4. 「保存」をクリック

## よくある質問

**Q: チャットボットは無料ですか？**
A: チャットボット機能自体は無料です。ただし、AIの回答にはAPIキー（AIプロバイダー）の利用料金がかかります。

**Q: 間違った回答をしませんか？**
A: 登録された情報のみをもとに回答します。情報にない質問には「お問い合わせください」と案内します。

**Q: 後からOFFにできますか？**
A: はい。設定画面からいつでもON/OFFを切り替えられます。

**Q: チャットボットのデザインは変えられますか？**
A: 現在はデフォルトデザインです。今後カスタマイズ機能を追加予定です。
