# ページ編集の使い方

## このステップで行うこと
「ページ」タブを使って、既存サイトの各ページのメタ情報を確認・編集します。

## ページタブの概要
サイトを接続すると、フォルダ内のすべてのHTMLファイルが「ページ」タブに一覧表示されます。ここから各ページのタイトルやdescriptionなどのメタ情報を管理できます。

## ページ一覧に表示される情報
| 項目 | 説明 |
|------|------|
| **ページパス** | ファイルの場所（例: `/services/ai-prediction/index.html`） |
| **タイトル** | `<title>`タグの内容 |
| **最終更新日** | ファイルの更新日時 |

## 操作手順

### ページのメタ情報を編集する
1. 「ページ」タブを開く
2. 編集したいページの行をクリック
3. 編集パネルが開く
4. 以下の項目を編集できます：
   - **ページタイトル** — `<title>`タグに反映
   - **meta description** — 検索結果に表示される説明文
   - **OGP設定** — SNSシェア時のタイトル・説明・画像
5. 「保存」をクリック

### ページをプレビューする
1. ページ一覧で対象ページの「👁 プレビュー」ボタンをクリック
2. ブラウザでページが表示されます

### 外部エディタで開く
1. ページ一覧で「📝 エディタで開く」をクリック
2. VS Codeなどの設定済みエディタでHTMLファイルが開きます

## メタ情報の反映について
ページタブで編集したメタ情報は、「設定反映（Inject）」を実行すると実際のHTMLファイルに書き込まれます。編集しただけでは、まだHTMLには反映されていません。

反映の流れ：
1. ページタブでメタ情報を編集・保存
2. 「検証・デプロイ」タブで「設定反映 (Inject)」を実行
3. HTMLファイルのmetaタグが更新される

## よくある質問

### Q. 新しいページを追加できますか？
現在のバージョンでは、ページタブからの新規ページ作成には対応していません。HTMLファイルを直接フォルダに追加すると、次回読み込み時にページ一覧に表示されます。

### Q. HTMLを直接編集してもいいですか？
はい、外部エディタで直接HTMLを編集できます。ただし、Site Builderが管理するmetaタグ領域（`<head>`内の特定タグ）は、Inject実行時に上書きされる場合があります。

### Q. カスタムコードは消えませんか？
Site Builderは「最小差分」方式でメタタグを更新します。独自に追加したHTMLやJavaScriptはそのまま保持されます。ただし、`<title>`や`<meta name="description">`など管理対象タグは上書きされます。
