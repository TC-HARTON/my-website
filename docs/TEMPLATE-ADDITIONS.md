## サンプルサイト作成時の注意

- 各業種のサンプルは `harton/samples/{id}/index.html` に配置
- SPEC v2.3 + GEO/LLMO準拠（Sランク品質 / 649項目検証）
- 業種特有のJSON-LD構造データを正確に設定
- FAQ は業種に合った実用的な質問を5問以上
- テーマカラーは上記 `color` を使用
- サンプル完成後、アプリの `TEMPLATES` 配列にエントリ追加
