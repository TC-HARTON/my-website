Claude API費用完全ガイド — Opus 4.6 公式価格準拠

公開: 2026-04-16 著者: 大内 達也

結論 — Claude API は「高くない」

本記事で判明する数値: LP 1枚 = ¥20（$0.14）、コーポレート 5ページ = ¥103（$0.68）、大規模 8P + ブログ10本 = ¥258（$1.72）。外注制作（¥100,000〜¥800,000）と比較して約 0.02〜0.1%の費用で同等以上の品質が得られます。

「Claude Opus 4.6 の公式価格は入力 $5/M tokens、出力 $25/M tokens。プロンプトキャッシュ利用時は入力が $0.50/M（90%割引）まで下がる」

— Anthropic Claude Opus 公式プロダクトページ

Claude Opus 4.6 公式価格の内訳

Anthropic 公式 Pricing に記載された 2026-04-16 時点の従量課金は以下の通りです（1 USD = 150 円換算）。

  • 入力トークン: $5 / 100万トークン（約¥750）
  • 出力トークン: $25 / 100万トークン（約¥3,750）
  • キャッシュ Hit 時の入力: $0.50 / 100万トークン（0.1x倍率 = 90%割引）
  • キャッシュ Write（5分）時の入力: $6.25 / 100万トークン（1.25x倍率、初回のみ）
  • キャッシュ Write（1時間）時の入力: $10 / 100万トークン（2x倍率、長時間キャッシュ）

※ 旧 Opus 4.1／4.0 は入力 $15／出力 $75 でしたが、Opus 4.5 以降は 1/3 に値下げされました。当記事はすべて Opus 4.6 最新価格で試算しています。

3法規プロンプト（40KB）の費用インパクト

Site Builder はシステムプロンプトに SPEC v2.4 + GOOGLE-STANDARDS + GEO-STANDARDS の計 約 40KB（約12,000トークン）を常時埋込みます。これを素で送ると1回 $0.06（約¥9）ですが、Anthropic のプロンプトキャッシュを使えば2回目以降は $0.006（¥0.9）に圧縮されます。

実運用での節約術

  • routingMode=balanced: 検証タスクを Haiku 4.5（入力$1/M・出力$5/M）にフォールバック → 費用 -60〜80%（詳細は routingMode完全ガイド）
  • プロンプトキャッシュ: 3法規40KBを ephemeral cache で -90%（仕組み解説）
  • 並列生成: ページ単位で並列化し体感速度 -70%
  • バッチ処理: Anthropic Batch API で入出力とも -50%（Opus 4.6: 入力 $2.50/M / 出力 $12.50/M）

使いすぎが心配な方へ — 月額上限設定

Anthropic コンソールで月額上限（Usage Limit）を $5〜$50 の任意額に設定できます。上限到達でAPIは自動停止し、追加請求は発生しません。$5 上限でも LP 約35枚分（旧 Opus 4.1 価格換算の12枚から大幅増）の作業が可能です。さらに routingMode を balanced に切り替えれば費用を -60〜80% 削減可能（詳細は API課金の安全装置ガイド）。

APIキーの取得

Claude APIキーは console.anthropic.com/settings/keys で無料発行できます。1本のキーで Opus / Sonnet / Haiku すべてにアクセス可能です（モデル別キーは不要）。詳細手順は Claude APIキー取得完全ガイド を参照してください。

よくある質問（FAQ）

Anthropic アカウント開設の初期費用はかかりますか？

アカウント作成とAPIキー発行は無料です。初回サインアップ時は $5 の無料クレジットが付与される場合があります。

実測と見積の誤差はどれくらい？

±15% 程度。事業情報の文字数やサービス件数で変動しますが、LP 1枚が ¥30 を超えることは通常ありません。

為替変動の影響は？

1 USD = 140〜160円の変動幅で、最大 +10% 程度の影響。常に円高方向の見積で計算しているため超過リスクは低いです。

Opus 4.1 から 4.6 に移行すると何が変わりますか？

APIキーの再発行は不要。リクエスト時の model パラメータを claude-opus-4-6 にするだけで、入力・出力単価ともに1/3（-67%）に即座に切り替わります。

まとめ

Claude Opus 4.6 の公式価格は 入力 $5/M・出力 $25/M。規模別実測で LP 1枚 ¥20 / コーポレート5P ¥103 / 大規模 ¥258。外注比で 0.02〜0.08% の驚異的コスパです。プロンプトキャッシュ（-90%）と routingMode=balanced（-60〜80%）を併用すれば、さらに大幅な削減が可能です。
