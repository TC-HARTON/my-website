AIチャットボット設置ガイド — 24時間自動対応

公開: 2026-04-12 著者: 大内 達也
Site Builder ガイド #Site Builder #初心者向け #チャットボット

AIチャットボットとは？

Site Builderに搭載されているAIチャットボットは、サイト訪問者の質問に24時間365日自動で回答する機能です。あなたが登録した事業情報・サービス内容・FAQをもとに、AIが的確な回答を生成します。営業時間外でも顧客対応ができるため、機会損失を防げます。

チャットボットでできること

  • 営業時間外の自動対応 — 深夜や休日でも訪問者の質問に即座に返答します
  • よくある質問への即時回答 — 登録したFAQ情報をもとに、正確な回答を自動生成します
  • サービス内容の案内 — 登録したサービス情報をもとに、適切なサービスを訪問者に紹介します
  • 問い合わせフォームへの誘導 — 詳細な相談が必要な場合は、お問い合わせフォームへスムーズに案内します

設定方法

  1. Site Builderの設定画面でチャットボットのON/OFFトグルを切り替えます
  2. 挨拶メッセージ（グリーティング）をカスタマイズします。例：「こんにちは！何かお手伝いできることはありますか？」
  3. 必要に応じてプロキシURL（上級者向け）を設定します。通常はデフォルトのままでOKです
  4. サイト生成を実行すると、チャットウィジェットが自動的にサイトに組み込まれます

回答の精度を高めるコツ

チャットボットの回答品質は、登録情報の充実度に比例します。以下のポイントを意識して情報を充実させましょう。

  • 事業情報をできるだけ詳しく入力する（サービス内容、対応エリア、強みなど）
  • FAQを最低5件以上登録し、実際によく聞かれる質問をカバーする
  • サービス説明を具体的に書き、料金や納期などの数字も含める

費用について

チャットボット機能自体は追加料金なしの無料機能です。ただし、AI の応答には Anthropic Claude API の従量課金が発生します（月数百円〜、訪問者数に比例）。チャットボット応答も設定済みの routingMode に従い、Haiku にフォールバックする routingMode=economy 設定なら月 100 円程度に抑えられます。

よくある質問（FAQ）

チャットボットの設置は必須ですか？

いいえ、完全なオプション機能です。不要な場合はOFFにしておけば、サイトにチャットウィジェットは表示されません。

回答内容を自分でカスタマイズできますか？

FAQや事業情報を充実させることで、間接的に回答内容をコントロールできます。AIが登録された情報をもとに、最適な回答を動的に生成します。

チャットボットの見た目は変更できますか？

挨拶メッセージのカスタマイズが可能です。ウィジェット自体のデザインはサイト全体のテーマカラーに合わせて自動調整されるため、統一感のある見た目になります。

まとめ

AIチャットボットを設置すれば、24時間365日の顧客対応が実現します。設定はON/OFFの切り替えとメッセージの入力だけで完了。Site Builderなら、登録済みの事業情報やFAQを最大限に活用して、賢いチャットボットが訪問者をサポートします。
