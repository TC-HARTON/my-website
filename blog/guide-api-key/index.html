Claude APIキー取得完全ガイド — 1本で Opus/Sonnet/Haiku すべて

公開: 2026-04-16 著者: 大内 達也
Claude API #Claude #APIキー #初心者向け

結論 — APIキーは1本で全モデル共通

本記事で判明する事実: Anthropic Claude の API キーは 1本で Opus 4.6・Sonnet 4.6・Haiku 4.5 すべてにアクセス可能です。モデル別に個別のキーを発行する必要はありません。Opus 専用ページ や Sonnet 専用ページ はモデル紹介用であり、それぞれに「別のキー」が必要というわけではありません。API リクエスト時に model パラメータで使い分けます。

「You can specify the model you want to use in your request body with the model parameter. Available models include claude-opus-4-6, claude-sonnet-4-6, and claude-haiku-4-5. The same API key authenticates all requests.」

— Anthropic API リファレンス: model parameter

取得手順（5分で完了）

  1. console.anthropic.com にアクセス — Google アカウントまたはメールアドレスでサインアップ可能
  2. メール認証を完了 — 確認メールの URL をクリック
  3. 組織情報を登録 — 個人利用なら「Personal」でOK
  4. 支払い方法を登録 — Settings → Plans & Billing → クレジットカード追加（初回 $5 無料クレジットが付与される場合あり）
  5. API キーを発行 — Settings → API Keys → Create Key をクリック
  6. キー名を入力（例: site-builder-main）して作成
  7. 表示された sk-ant-api03-... をコピー — 再表示できないため、この場で保管場所に貼り付け
  8. Site Builder の設定画面に貼り付け → 接続テストで動作確認

※ 発行後は Usage ダッシュボード でリアルタイムの利用額が確認できます。

APIキーのフォーマット

sk-ant-api03-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxAA
  • プレフィックス: sk-ant-api03-（2026年4月現在のバージョン）
  • 長さ: 約 100 文字の英数字＋ハイフン
  • 用途: HTTP Header x-api-key: sk-ant-api03-... で認証
  • 有効期限: 明示的な失効日なし（Revoke するまで永続）

1本のキーでモデルを切り替える仕組み

Anthropic の API は REST エンドポイント https://api.anthropic.com/v1/messages に対して model パラメータを変えるだけで、同一キーから各モデルへリクエストできます。

// 共通の HTTP ヘッダー（3 モデル共通・キーは 1 本）
//   x-api-key: sk-ant-api03-...
//   anthropic-version: 2023-06-01
//   content-type: application/json

// Opus 4.6 を使う場合
{
  "model": "claude-opus-4-6",
  "max_tokens": 4096,
  "messages": [{"role": "user", "content": "Hello"}]
}

// Sonnet 4.6 を使う場合（同じキー・同じエンドポイントで model だけ差し替え）
{
  "model": "claude-sonnet-4-6",
  "max_tokens": 4096,
  "messages": [{"role": "user", "content": "Hello"}]
}

// Haiku 4.5 を使う場合（同じキー・同じエンドポイントで model だけ差し替え）
{
  "model": "claude-haiku-4-5-20251001",
  "max_tokens": 4096,
  "messages": [{"role": "user", "content": "Hello"}]
}

※ 各モデルの正式な model 文字列は Anthropic 公式 Models ドキュメント で最新のものをご確認ください。

月額上限の設定（必須）

発行直後に必ず Settings → Limits で月額上限を設定してください。上限到達時に API が自動停止し、想定外の課金を防ぎます。

  • $5 上限 — 個人利用の目安（LP 約35枚 / コーポレート5P 約7サイト / ブログ約31本）
  • $20 上限 — 小規模事業者の目安
  • $50 上限 — 制作会社の月5件ペース

詳しくは API課金の安全装置ガイド を参照してください。

セキュリティの鉄則

  • クライアントサイドに埋め込まない — JavaScript から直接呼ぶと誰でもキーを盗めます。Site Builder は AES-256 ローカル暗号化で保存
  • Git リポジトリにコミットしない — .env / .gitignore を必ず設定。うっかりコミットしたら即 Revoke
  • 公開チャットに貼らない — AI チャット含め、チャット履歴に流れると流出リスク
  • 定期ローテーション — 半年〜1年ごとに Revoke → 新規発行を推奨
  • 万一の漏洩時 — Keys 画面 で即 Revoke（無効化）可能。既存生成物への影響はありません

Claude MAX プラン（$100/月）との違い

Anthropic には有料サブスクリプションの Claude MAX プランも存在しますが、これは claude.ai（ブラウザのチャットUI）を大量に使うためのプランで、API 利用とは完全に別サービスです。

よくある質問（FAQ）

Opus 用と Sonnet 用で別々にキーが必要ですか？

いいえ、1本で十分です。Opus 公式ページや Sonnet 公式ページはモデルの機能紹介で、それぞれに別のキーを発行する仕組みではありません。API リクエスト時の model パラメータで切り替えます。

キーを忘れてしまいました

再表示はできません。Keys 画面で古いキーを Revoke し、新しいキーを発行して Site Builder の設定を更新してください。

クレジットカードなしで試せますか？

アカウント作成とキー発行は無料ですが、API 呼び出しには支払い方法の登録が必須です。初回 $5 の無料クレジットが付与されるケースがあり、その範囲内なら実質無料で試せます。

法人契約は可能ですか？

はい、Organization 単位での契約が可能です。Settings → Organization で複数メンバー招待・請求先分離が可能で、企業のセキュリティ要件に沿った管理ができます。詳細は Anthropic Enterprise を参照してください。

Gemini や ChatGPT のキーを流用できますか？

いいえ、できません。Anthropic Claude / Google Gemini / OpenAI GPT はそれぞれ独立した会社・認証基盤のサービスです。Site Builder は Anthropic Claude 専用設計のため、sk-ant-api03-... 形式のキーのみ動作します。

まとめ

Claude API キーは Anthropic Console で無料発行でき、1本で Opus 4.6・Sonnet 4.6・Haiku 4.5 すべてにアクセス可能です。発行後は必ず月額上限（$5〜）を設定し、セキュリティ鉄則（クライアント埋込禁止・Git 管理外・漏洩時は即 Revoke）を守りましょう。費用の全体像は Claude API費用完全ガイド、キャッシュ活用は プロンプトキャッシュ完全解説、課金安全装置は API課金の安全装置ガイド を参照してください。
