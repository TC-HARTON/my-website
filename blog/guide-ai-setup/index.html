Claude API の設定方法 — Sクラス保証の唯一条件

公開: 2026-04-16 著者: 大内 達也
Site Builder ガイド #Site Builder #初心者向け #AI設定

なぜ Claude Opus 4.6 限定なのか

「Generative Engine Optimization (GEO) は、生成AIの引用対象として採用されるための最適化手法であり、Quotation・Statistics・Citation の3要素で引用率が平均 40.6% 改善する」

— Aggarwal et al. (Cornell Univ.) Generative Engine Optimization, KDD 2024

HARTON Site Builder は 2026年4月より Claude Opus 4.6 限定に刷新しました。これは、以下の3法規に完全準拠してサイト生成できる唯一のモデルだからです。

  1. SPEC v2.3 — 313項目の自社品質規格（日本語E-E-A-T・構造化データ・アクセシビリティ等）
  2. GOOGLE-STANDARDS — Google公式「有用で信頼性の高いコンテンツ」ガイドライン
  3. GEO-STANDARDS — Cornell大学 KDD2024 論文（arXiv:2311.09735）の生成エンジン最適化 9手法

Gemini / ChatGPT は日本語 E-E-A-T 評価が劣後するため、Sクラス保証の対象外となりました（サポート終了）。

モデル対応表

  • Claude Opus 4.6 ✅ Sクラス保証（推奨） — 3法規完全準拠
  • Claude Sonnet 4.5 ⚠ Aクラス想定 — 分業ルーティング時のサブモデル
  • Claude Haiku 4 ⚠ 検証・軽量処理用
  • Gemini / GPT ❌ 非対応 — 日本語 E-E-A-T 劣後により Sクラス保証不可

APIキーの取得手順

  1. console.anthropic.com/settings/keys にアクセス
  2. Anthropicアカウントでログイン（未作成ならサインアップ）
  3. 「Create Key」から新しいAPIキーを発行
  4. 表示された sk-ant-api... 形式のキーを必ずコピー（再表示不可）
  5. Site Builder 設定画面「Claude API」セクションに貼り付け
  6. 「接続テスト」で動作確認

ルーティング戦略の選択（routingMode）

Site Builder はモデル分業ルーティングに対応し、用途別に最適なモデルを自動選択します。

  • quality（全 Opus 推奨） — 設計・生成・検証すべて Opus 4.6。最高品質。月2〜3サイト生成者向け
  • balanced — サイト構造設計=Opus、HTMLコーディング=Opus、検証=Haiku の分業。コストと品質のバランス最適
  • economy — 反復作業を Haiku にフォールバック。月5サイト以上の大量生成者向け

プロンプトキャッシュによる費用削減

3法規の仕様書 約40KBがシステムプロンプトに常時埋込まれますが、Anthropicのプロンプトキャッシュ機能により2回目以降の生成は最大85%のトークン費用削減が可能です。

費用目安

  • 従量制（balanced モード） — LP 1枚 約¥60、コーポレート5P 約¥310
  • 月額上限設定 — Anthropic コンソールで $5〜 の上限を設定可能。上限到達で自動停止（安全装置の詳細）
  • 外注（Web制作会社） — ¥300,000〜が一般相場。Site Builderなら同等品質で 0.1%以下 の費用

セキュリティ（BYO-Key）

APIキーはお使いのPCにAES-256ローカル暗号化で保存されます。アプリ運営側のサーバーを経由しないBYO-Key（Bring Your Own Key）方式のため、第三者にキーが漏れる経路は存在しません。万一漏洩した場合は Anthropic Console から即座にRevoke可能です。

よくある質問（FAQ）

Anthropicアカウントの開設は有料ですか？

アカウント作成とキー発行は無料です。初回は $5 の無料クレジットが付与される場合もあります。実際の利用は従量課金で、サイト1本あたり数百〜千円程度です。

SonnetやHaikuだけで生成できますか？

可能ですが Sクラス保証の対象外（Aクラス想定）となります。サイト構造設計など中核部分は Opus 4.6 の使用を強く推奨します。

既存の Gemini/GPT キーは使えますか？

2026年4月の刷新以降、Gemini / ChatGPT は非対応になりました。Anthropic Claude API キーへの移行をお願いします。

まとめ

Sクラス品質保証の唯一条件は Claude Opus 4.6 の利用です。APIキーは Anthropic Console で発行し、Site Builder に貼り付けるだけ。プロンプトキャッシュと分業ルーティングにより、最高品質と低コストを両立できます。
